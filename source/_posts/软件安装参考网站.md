# sublime text 3

* 教程：[命令行安装sublime text 3](https://jingyan.baidu.com/article/64d05a023cd849de55f73be4.html)

* 概括：三条命令即可
    ```
    sudo add-apt-repository ppa:webupd8team/sublime-text-3
    sudo apt-get update
    sudo apt-get install sublime-text-installer
    ```
